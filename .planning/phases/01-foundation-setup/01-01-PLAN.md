<task type="auto">
  <name>Task 2: Configure TypeScript and build optimization</name>
  <files>tsconfig.json, tsconfig.app.json, vite.config.ts</files>
  <action>
Configure TypeScript for strict type checking and optimize Vite build settings.

Update tsconfig.json:
- Set "compilerOptions.target": "ES2020"
- Set "compilerOptions.lib": ["ES2020", "DOM", "DOM.Iterable"]
- Set "compilerOptions.module": "ESNext"
- Enable "compilerOptions.strict": true
- Enable "compilerOptions.noUnusedLocals": true
- Enable "compilerOptions.noUnusedParameters": true
- Enable "compilerOptions.noFallthroughCasesInSwitch": true
- Set "compilerOptions.moduleResolution": "bundler"
- Set "compilerOptions.allowImportingTsExtensions": true
- Set "compilerOptions.resolveJsonModule": true
- Set "compilerOptions isolatedModules": true
- Set "compilerOptions.noEmit": true
- Set "compilerOptions.jsx": "react-jsx"

Update vite.config.ts:
- Add esbuild: { logLevel: 'warning' } for cleaner build output
- Keep default settings (everything else is optimized)

Create tsconfig.app.json for app-specific config extending base tsconfig:
{
  "extends": "@tsconfig/react-tsconfig/tsconfig.json",
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  },
  "include": ["src"]
}

This establishes @/* path alias for cleaner imports.
  </action>

- npm run build completes successfully
- No TypeScript errors in src/App.tsx or src/main.tsx
- @ alias can be used in imports (test with a simple import if needed)
  </verify>
  <done>
